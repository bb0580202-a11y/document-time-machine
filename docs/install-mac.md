# 在 Mac 上安装「文档时光机」

> 仅支持 **Apple 芯片 Mac**（M 系列）。Intel Mac 暂不支持。

## 1. 装进电脑
1. 双击下载到的 `doc-time-machine.dmg`，会弹出一个窗口。
2. 把左边的**应用图标**，拖到右边的 **Applications（应用程序）** 文件夹上。
3. 拖完就装好了，可以把这个窗口关掉。

## 2. 第一次打开
打开「应用程序」，**双击** doc-time-machine。

- **如果直接开起来了**（比如你是用**微信**收到的文件）：太好了，跳到第 3 步。
- **如果弹出「无法打开，因为无法验证开发者」**（比如你是从**网盘/浏览器**下载的）：这不是病毒，是因为这是免费开源软件、没买苹果的签名。跟着做一次就好，以后不再拦：
  1. 那个弹窗点**完成/取消**（先别管）。
  2. 打开 **系统设置 → 隐私与安全性**，往下滚，会看到一行
     「已阻止 doc-time-machine…」，点它右边的「**仍要打开**」。
  3. 再弹一次确认，点「**仍要打开**」，输入你的**开机密码**。
  4. 这次就开起来了。

开起来后，顶部菜单栏（右上角）会出现一个小圆点图标 = 它在后台帮你看管文件了。

## 3. 开始用
点菜单栏小圆点 →「打开主界面」→「添加文件夹」→ 选你写论文的文件夹。
以后这个文件夹一有改动，它就自动帮你存一版，写崩了能找回任何一天的样子。

> 文案以 macOS 26 (Tahoe) 真机两渠道（微信/网盘）走通核对。配图可后续补（不阻塞使用）。
