# 怎么找回某天的稿子

写崩了、删多了、想看看上周那版怎么写的——跟着做就行。**整个过程不会动你现在正在用的文件**，找回的是另存的一份副本，放心操作。

## 1. 打开主界面
点屏幕**右上角菜单栏**那个**蓝色小圆点**图标 →「打开主界面」。

## 2. 在相册里找到那一天
中间的**相册**把每一次自动存档按时间排好了，**最新的在最上面**。每张卡片写着：
- 存档的**时间**（几月几号几点，以及"3 分钟前 / 昨天"这种）；
- 如果你给某一版起过名字（比如"投稿前"），卡片会显示那个名字；
- 这一版**改了哪个文件**。

往下翻，找到你想回到的那一天那一版。
（左边的**版本树**是一张时间地图，版本多的时候帮你快速定位；用不上可以点「⊟ 版本树」收起来。）

## 3. 还原这一版
点那张卡片上的「**还原这一版**」→ 会先问你一句确认 → 点「确定还原」。

## 4. 在文件夹旁边拿到副本
还原完，底部会弹一条绿色提示：**已另存到文件夹旁边：xxx_恢复自….docx**。
- 点提示里的「**📂 在访达中打开**」，就直接跳到那个副本；
- 或者自己打开那个文件夹，副本就在你原文件**旁边**，名字带「恢复自+日期」。

打开副本看看，是不是你要的那天的样子。**你原来的文件一直没动**——副本是额外多出来的一份，确认没问题再决定怎么用。

---

> 小提示：你正常写、正常按 Ctrl+S 就行，它每次保存都默默存下一版——**你存过的，任何一版都随时回得去**。
