# 文档时光机（doc-time-machine）

给写论文的人做的"后悔药"：自动给你的 Word / Excel / PPT / PDF 文件夹存档，
**改坏了、软件崩了、想退回旧版本**，都能找回来。

- **纯本地**：所有版本都存在你自己的硬盘上，不联网、不上传、不依赖任何账号。
- **只增不减**：只往历史里加存档，从不动你正在编辑的文件。
- **会搬家**：版本历史就藏在这个文件夹里，整个文件夹拷到 U 盘 / 新电脑，历史跟着走。

> 命令行 + 图形界面 / 托盘后台 / 开机自启 + 打包双击即装（**Mac 与 Windows 均已可用**，内置 git 无需另装）。

## 安装

最新版在 [**Releases**](../../releases/latest) 下载：

- **Mac（Apple 芯片）**：`doc-time-machine.dmg`（约 23M）双击安装，照 [`docs/install-mac.md`](docs/install-mac.md) 走一次首次打开。怎么用见 [`docs/找回某天的稿子.md`](docs/找回某天的稿子.md)。
- **Windows（x64）**：`doc-time-machine-windows-x64.zip` 解压**整个文件夹**，双击里面的 `doc-time-machine.exe`（别只拷 .exe，要连 `_internal` 一起）。

> 未签名：Mac 首开「右键→打开」或「设置→仍要打开」；Windows SmartScreen 拦就点「更多信息→仍要运行」。

**从源码运行 / 开发**：需先装好 [git](https://git-scm.com/) 和 Python 3.10+：

```bash
pip install -e .            # 跑 CLI / 从源码起 GUI（python -m dtm.app.daemon）
bash scripts/build_mac.sh --dmg   # 一键打包出 Mac .app + .dmg（需 create-dmg）
```

## 用法

| 想做什么 | 命令 |
|---|---|
| 给一个文件夹开启守护 | `dtm init <文件夹>` |
| 开始看着它、自动存档 | `dtm watch <文件夹>`（前台运行，Ctrl+C 停止）|
| 看有哪些历史版本 | `dtm list <文件夹>` |
| 把某个旧版本找回来（存到旁边，不覆盖当前） | `dtm restore <文件夹> <版本号> <文件名>` |
| 给重要版本起个名（里程碑） | `dtm tag <文件夹> <版本号> "投稿前"` |
| 给某个版本随手写句备注（便利贴） | `dtm note <文件夹> <版本号> "导师说结论太弱"` |
| 从某个旧版本另开一条写作路线 | `dtm branch <文件夹> <版本号>` |
| 看历史占了多大、有多少版本 | `dtm stats <文件夹>` |
| 检查文件是否完好能打开 | `dtm verify <文件夹>` |
| 压缩一下历史占用 | `dtm gc <文件夹>` |
| 文件夹搬过家后重新认领历史 | `dtm relocate <文件夹>` |

版本号就是 `dtm list` 里每行开头方括号里那串短编号。

## 图形界面 / 后台守护（Phase 2）

```bash
dtm daemon              # 启动托盘后台守护：自动看护已登记的文件夹 + 菜单栏图标
dtm autostart enable    # 设为开机自启（macOS launchd）；disable 关闭、status 查看
```

托盘「打开主界面」开窗：

- **相册**（主视图）：每个版本一张卡 —— 备注当标题（无备注回落到时间）、逐文件 ▲变大/▼变小、★里程碑、悬停看精确到秒。三个动作：**还原这一版**（另存一份到文件夹旁边，绝不覆盖你当前的文件）/ 加备注 / 标里程碑。
- **版本树**（侧边地图）：分段时间轴（长空白压缩成"≈无改动N天"、突发密集聚合成簇）、多条写作路线分泳道（其余折叠成侧边条、只露最新一版）、点地标联动右侧相册。仅当真有分叉时默认展开。
- 后台新存的版本，窗口几秒内**自动刷新**（也有手动「↻ 刷新」）。
- 顶栏还能：**📂 打开文件夹**（在访达里定位当前库）、**✕ 移除**（不再看管，文件和历史都留着、可一键撤销）；还原后弹持久提示，一键在访达里定位还原出的副本。
- **安全提醒（Phase 3 robustness）**：仓库若损坏（如断电打断保存）顶部出**红条**告警；守护若曾意外停过，下次开窗出**黄条**告诉你"刚才有段没被保护"——绝不让你在不知情下裸奔。

## 它存什么、不存什么

- **重点保护**：`.docx .xlsx .pptx .pdf .doc .xls .ppt .tex .md .txt .csv`
- **自动忽略垃圾**：Office 临时锁文件（`~$…`）、`.tmp`、`.DS_Store` 等
- **拿不准的文件**：一律先存上（宁可多存垃圾，绝不漏掉你的正文）

## 诚实的边界：这些情况它救不了（请先看）

它专治**"改坏了 / 误删了 / 想退回旧版本"**。但有几类问题，**任何纯本地软件都救不了**，下载前请务必知道（详见 [`docs/这些情况它救不了.md`](docs/这些情况它救不了.md)）：

- **硬盘物理坏了 / 整盘丢了 / 电脑被偷、进水** —— 历史就藏在这块盘里，盘没了它也没了。这类**只有"再拷一份到别处"能防**：请偶尔把整个文件夹拷进移动硬盘 / 网盘备一份（你拷走的副本自带全套历史）。
- **硬盘坏块导致某一版历史数据烂掉** —— **那一版可能找不回**（但其它版本通常不受影响）。同样靠"偶尔另拷一份"兜底。
- **源文件本身被写坏了**（Word/PDF 保存时就崩了）—— 它**忠实存下你电脑上的样子，修不了已经坏掉的文件**；但你之前的正常版本都在，可随时还原回去。它会在备份时**如实告诉你这一版可能是坏的**，绝不假装修好了。

其它边界：

- Office 是二进制文件，能告诉你"哪个文件、哪一版、变大变小"，但**看不清相邻两版具体改了哪句话**。
- 常改的大文件版本会占空间，可随时 `dtm gc` 压缩；更彻底的老版本瘦身留给后续版本。
